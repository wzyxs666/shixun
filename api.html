MQTT 协议接口

基础配置

参数 默认值 说明 服务器地址 192.168.1.100 MQTT Broker IP 地址 端口 1883 MQTT 服务端口 客户端 ID PCT_100_XXX 设备唯一标识符 发布主题 chemctrl/{Device_ID}/status 状态上报主题 订阅主题 chemctrl/{Device_ID}/command 命令接收主题

状态上报

PUBLISH chemctrl/{Device_ID}/status

上报频率

  • 定时上报：每 30 秒
  • 事件触发：状态变更、阈值穿越时立即上报

Payload 格式

{
  "device_id": "PCT_100_111",
  "temperature": 25.5,
  "light": 320,
  "mode": "auto",
  "key1_state": true,
  "relay_state": 2,
  "temp_threshold": 30.5,
  "light_threshold": 260,
  "timestamp": 1672531200
}

字段说明

字段 类型 说明 device_id string 设备唯一标识 temperature float 当前温度（°C） light int 光照强度（lm） mode string 运行模式：auto / manual key1_state bool 总闸状态：true=合上 relay_state int 继电器状态（0-3） temp_threshold float 温度阈值（°C） light_threshold int 光照阈值（lm） timestamp int 时间戳（秒）

命令控制

SUBSCRIBE chemctrl/{Device_ID}/command

支持的命令

1. set_relay - 设置继电器状态

{
  "cmd": "set_relay",
  "relay": 3,
  "state": true
}
参数 类型 说明 relay int 继电器编号：3=灯光，4=风扇 state bool 状态：true=开启，false=关闭

2. set_mode - 设置运行模式

{
  "cmd": "set_mode",
  "mode": "manual"
}
参数 类型 说明 mode string 模式：auto / manual

3. set_threshold - 设置阈值

{
  "cmd": "set_threshold",
  "type": "temp",
  "value": 32.0
}
参数 类型 说明 type string 阈值类型：temp / light value float 阈值数值

4. get_status - 查询状态

{
  "cmd": "get_status"
}

设备收到此命令后，立即上报当前状态到发布主题。

5. reboot - 远程重启

{
  "cmd": "reboot"
}

设备延迟 1 秒后执行软复位。

串口命令

系统命令

命令 功能 示例 wifi 进入 Wi-Fi 扫描配网模式 wifi wifion 连接默认 Wi-Fi wifion wifioff 断开 Wi-Fi 连接 wifioff clearwifi 清除保存的 Wi-Fi 配置 clearwifi ip 显示网络信息 ip status 显示系统状态 status mode 切换运行模式 mode help 显示帮助信息 help

控制命令（手动模式）

命令 功能 说明 light 切换灯光 手动模式下有效 fan 切换风扇 手动模式下有效 off 关闭所有设备 立即关闭灯光和风扇

MQTT 配置命令

命令 功能 示例 setmqip 设置 MQTT 服务器 IP setmqip 192.168.1.100 setmqport 设置 MQTT 端口 setmqport 1883 setmquser 设置用户名 setmquser admin setmqpass 设置密码 setmqpass password setmqid 设置设备 ID setmqid PCT_100_001 showmqtt 显示 MQTT 配置 showmqtt

RGB LED 命令

命令 功能 r / rgb_red 设置红色 g / rgb_green 设置绿色 b / rgb_blue 设置蓝色 y / rgb_yellow 设置黄色 p / rgb_purple 设置紫色 c / rgb_cyan 设置青色 w / rgb_white 设置白色 o / rgb_off 恢复自动状态指示 rainbow 彩虹循环动画（5秒） fade 呼吸灯效果（5秒） blink 闪烁效果（5秒） t / rgb_test 颜色测试（每种颜色5秒）
